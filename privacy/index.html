뭐먹지? 개인정보 처리방침

시행일: 2026년 6월 15일

뭐먹지?는 서비스 제공에 필요한 최소한의 정보만 처리합니다.

1. 처리하는 정보

  • 게스트 식사 기록: 음식 이름, 식사 구분, 즐겨찾기 여부, 기록 시각을 사용자 기기에 저장
  • 토스 로그인 식별자: 사업자 인증 후 계정 동기화를 제공하는 경우 사용자별 서버 기록 구분
  • 위치 정보: 날씨 조회에만 일시적으로 사용하며 서버에 저장하지 않음
  • 이용 이벤트: 추천 노출과 선택 흐름. 사용자 식별자와 직접 입력한 음식 이름은 포함하지 않음

2. 처리 목적

게스트 기기 저장, 제공되는 경우 토스 로그인, 식사 기록 관리, 메뉴 추천, 식사 통계와 서비스 이용성 개선을 위해 처리합니다.

3. 보유 및 이용 기간

게스트 식사 기록은 사용자의 기기에 저장되며 직접 삭제할 때까지 보관합니다. 계정 동기화 기록은 사용자가 삭제하거나 서비스 이용이 종료될 때까지 서버에 보관합니다. 앱 설정의 모든 식사 기록 삭제 기능으로 즉시 삭제할 수 있습니다.

4. 외부 서비스

  • 토스: 사업자 인증 후 계정 동기화를 제공하는 경우 로그인과 사용자 인증
  • Supabase: 인증 중계 API와 식사 기록 데이터베이스
  • Open-Meteo: 현재 위치 또는 서울 기준 날씨 조회
  • 앱인토스 Analytics: 익명화된 노출 및 클릭 이벤트 분석

5. 이용자의 권리

사용자는 앱에서 식사 기록을 삭제하거나 고객문의 이메일을 통해 열람, 정정, 삭제와 처리 정지를 요청할 수 있습니다.

6. 안전성 확보 조치

서버에서 토스 로그인 정보를 다시 검증하고, 사용자별 데이터 접근을 분리하며, 데이터베이스의 공개 익명 접근을 차단합니다.
