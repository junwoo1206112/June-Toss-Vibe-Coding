## Context

기존 meal-recommender 앱에 통계, 다중 추천, 로딩 상태 UX 개선을 추가합니다.

## Goals / Non-Goals

**Goals:** 통계 탭 (TOP5, 요일/시간대 패턴), 3개 추천 카드, 로딩 스켈레톤, 홈에서 로그아웃
**Non-Goals:** AI 분석, 소셜 공유, PDF 내보내기

## Decisions

| 결정 | 선택 | 이유 |
|------|------|------|
| 통계 연산 | 클라이언트 사이드 (Supabase에서 전체 데이터 조회 후 집계) | 데이터 양이 적고 Supabase 무료 티어 제한 고려 |
| 추천 개수 | 3개 | 과하지 않고 선택지 제공 |
| 로딩 UI | 순수 CSS 애니메이션 스켈레톤 | 라이브러리 추가 없이 구현 |
| 로그아웃 | 홈 화면 상단에 작은 로그아웃 아이콘 | 설정까지 가는 불편 해소 |

## Risks / Trade-offs

| Risk | Mitigation |
|------|-----------|
| 통계 페이지에서 전체 데이터 로드 시 무거움 | 통계는 최근 90일로 제한 |
| 3개 추천 시 동일한 메뉴 중복 | 중복 제거 로직 적용 |
